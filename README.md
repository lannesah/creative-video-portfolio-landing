# creative-video-portfolio-landing
A one-page portfolio built for a video editor, focused on storytelling, visual clarity and real-world presentation.

This project is a one-page portfolio developed for a video editor, with a focus on visual storytelling, clarity and real-world presentation.

The goal was to create a clean and modern experience that highlights creative work without unnecessary complexity, balancing strong visuals with simple and efficient structure.

---

## Project Overview

The layout follows a narrative flow:

- Introduction with a strong visual impact
- Presentation of skills and creative approach
- Showcase of selected video projects
- Supporting sections (experience, tools and testimonials)
- Final call to action

Each section was designed to guide the viewer naturally through the content, without overwhelming interactions or distractions.

---

## Key Features

- Clean and structured HTML5
- Organized CSS with scalable naming (BEM approach)
- Smooth scroll and subtle transitions
- Video showcase sections with interactive behavior
- Responsive layout for different screen sizes
- Focus on readability and visual hierarchy

---

## Technologies

- HTML5
- CSS3
- JavaScript (vanilla)

No frameworks were used in order to keep the project lightweight and focused on fundamentals.

---

## Purpose

This project was developed as part of a personal study process, with the intention of:

- Improving front-end structure and organization
- Replicating real-world layouts with attention to detail
- Understanding how visual design translates into code
- Building a portfolio that can be used in real freelance scenarios
